# Chrome 测试数据生成器扩展

一个功能强大的 Chrome 浏览器扩展，用于快速生成各种测试数据。

## 功能特性

- ✅ **身份证号生成** - 生成符合中国身份证规则的有效身份证号
- ✅ **手机号生成** - 生成中国手机号码
- ✅ **邮箱生成** - 生成随机邮箱地址
- ✅ **中文姓名生成** - 生成随机中文姓名
- ✅ **银行卡号生成** - 生成符合 Luhn 算法的银行卡号
- ✅ **地址生成** - 生成随机地址
- ✅ **UUID 生成** - 生成标准 UUID
- ✅ **日期时间生成** - 生成随机日期时间
- ✅ **随机字符串生成** - 生成随机字符串
- ✅ **一键复制** - 快速复制到剪贴板
- ✅ **一键生成全部** - 批量生成所有类型数据

## 安装方法

### 方法一：开发者模式安装（推荐）

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `backend/chrome/test-data-generator` 目录
6. 扩展安装完成！

### 方法二：打包安装

**使用 Chrome 浏览器打包：**

1. 在 `chrome://extensions/` 页面点击"打包扩展程序"
2. 选择 `backend/chrome/test-data-generator` 目录
3. 生成 `.crx` 文件后，拖拽到 Chrome 浏览器安装

**使用打包脚本（推荐）：**

```bash
cd backend/chrome/test-data-generator
./package.sh
```

脚本会在上一级目录生成 `test-data-generator.zip` 文件。

📖 **详细打包说明请查看 [PACKAGE.md](./PACKAGE.md)**

## 使用方法

1. 点击浏览器工具栏中的扩展图标
2. 在弹出的窗口中：
   - 点击"生成"按钮生成对应类型的数据
   - 点击"📋"按钮复制数据到剪贴板
   - 点击"一键生成全部"批量生成所有数据
   - 点击输入框可自动选中文本

## 技术说明

### 身份证号生成算法

- 使用中国省份代码
- 生成有效出生日期
- 计算正确的校验码（最后一位）

### 银行卡号生成算法

- 使用常见银行卡前缀
- 应用 Luhn 算法生成有效校验码

### 数据生成规则

- 手机号：使用常见运营商前缀
- 邮箱：随机用户名 + 常见邮箱域名
- 姓名：常见姓氏 + 常见名字组合
- 地址：随机省份城市 + 街道 + 门牌号

## 注意事项

⚠️ **重要提示**：

- 本扩展生成的数据仅用于测试目的
- 生成的身份证号、银行卡号等数据是随机的，不代表真实存在
- 请勿将生成的数据用于任何非法用途
- 在生产环境中使用前，请确保符合相关法律法规
